# Installation

Download init script

```bash
wget https://raw.githubusercontent.com/UISSH/install-script/main/install.sh
bash install.sh
```

Start install

```bash
cd /tmp/install-script 
python3 main.py \
 --set_db_root_password={password} \
 --set_login_email={email} \
 --set_login_password={password}
```
